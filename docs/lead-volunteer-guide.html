Dashboard: st-mark-pantry.vercel.app

1. Getting Started

Opening the Dashboard

1. On your phone or Chromebook, open the browser and go to st-mark-pantry.vercel.app
2. Enter the password: stmark
3. You'll see the Dashboard with today's stats and quick action buttons
Tip: On your phone, tap "Add to Home Screen" (in the browser menu) to make it feel like a regular app. You only enter the password once per device.

Bottom Navigation Bar

Five tabs at the bottom of every screen:

TabWhat It Does
HomeDashboard — today's stats, recent check-ins, quick actions
ClientsView, search, add, and edit client records
Check-InCheck in clients during a pantry session
ReportsMonthly stats, inactive clients report
SettingsData backup, cloud sync, feature toggles

2. Before a Session

Adding New Clients

1. Tap Clients → Add Client (top right) — or use the green + button on the Check-In page
2. Enter: First Name, Last Name, Family Size (required)
3. Optional: phone, email, address, family members, notes
4. Set "Accepts Perishable Foods" to No if the client declines perishables
5. Tap Save

Reviewing the Volunteer Schedule

1. Tap Home → Volunteers quick action
2. Tap Schedule (top of volunteer list)
3. See who's signed up for upcoming sessions — each date shows the volunteers and their roles

3. During a Session — Client Check-In

This is the main workflow you'll use during each Monday, Friday, or Saturday session.

1. Tap Check-In in the bottom nav
2. Select the correct day pill: Monday, Friday, or Saturday (auto-detects if today is a pantry day)
3. Enter your name in the "Served by" field (optional, tracks who helped each client)
4. As each client arrives, type the first few letters of their name in the search box
5. Tap the green Check In button next to the client's name
6. The client appears in the "Today's Visitors" list at the bottom
Duplicate Warning: If a client already visited this month, you'll see a yellow warning. You can still check them in — just tap "Check In Anyway" to confirm.

If a Client Isn't in the System

1. Tap the green + button (bottom right of Check-In screen)
2. Fill in their info and save
3. Return to Check-In — search their name and check them in

4. Managing Volunteers

Adding a Volunteer

1. Tap Home → Volunteers
2. Tap Add (top right)
3. Enter name, email (for reminders), phone, and recurring schedule
4. Set recurring slots like "Every Monday" or "2nd Saturday" for regular volunteers
Note: Volunteers who sign up through the public website (st-mark-pantry.vercel.app/site/volunteer.html) are automatically saved to the cloud. Use Settings → Sync from Cloud to pull them into the dashboard.

Volunteer Check-In (During a Session)

1. Tap Home → Volunteers → Check In
2. Search the volunteer's name
3. Select their role (Intake, Distribution, Setup, Cleanup, etc.)
4. Tap Check In

5. After a Session

Reviewing Reports

1. Tap Reports in the bottom nav
2. See monthly client counts, visit totals, and trends
3. Tap Inactive Clients to see who hasn't visited this month

Backing Up Data

1. Tap Settings
2. Tap Export All Data (JSON) — saves a backup file to your device
3. To restore: tap Import Data (JSON) and select the backup file
Important: All data is stored on your device. If you clear browser data, it's gone. Export a backup regularly (weekly is ideal).

6. Cloud Sync

Pushing Volunteers to Cloud

This enables automated reminder emails 3 days before each session.

1. Tap Settings
2. Turn on Enable Notifications
3. Tap Sync All Volunteers to Cloud

Pulling from Cloud (New Public Signups)

1. Tap Settings
2. Ensure Enable Notifications is on
3. Tap Sync from Cloud
4. New volunteers and signups from the website will appear in the dashboard
